Connection Stack

Browser (index.html) ↓ JavaScript (dashboard-api.js) auto-initializes on page load ↓ Fetch API calls to PHP endpoints (dashboard.php, events.php, etc.) ↓ Database queries via db_config.php ↓ JSON response back to UI ↓ Charts & Tables rendered with real data

What Happens on Page Load

  • HTML page loads with all page containers
  • JavaScript dashboard-api.js initializes (event DOMContentLoaded)
  • DashboardManager.init() runs automatically
  • loadDashboardPage() called → fetches from dashboard.php API
  • API queries database for stats and trends
  • KPI cards update with: Total Events, Registrations, Check-ins, Upcoming
  • Chart.js initializes with real data (Registration Trends, Event Types)
  • Page is ready with all live data

Troubleshooting

If KPI cards show "0":
  • Check browser console (F12) for errors
  • Verify database has data in: events, registrations, users tables
  • Run test-api.html to check which API is failing
  • Check PHP error logs in XAMPP logs folder
Success Indicators:
  • ✓ KPI cards show numbers (not zero)
  • ✓ Charts render with data
  • ✓ Console shows no red errors
