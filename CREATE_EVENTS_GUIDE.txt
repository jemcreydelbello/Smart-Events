👥 CAPACITY MANAGEMENT:

The system automatically calculates:

  Available Spots = Total Capacity - Current Registrations

Example:
  Event Capacity: 100
  Current Registrations: 45
  Available Spots: 55

Display on Event Card:
  "Capacity: 100 | Available: 55"

📋 API ENDPOINT - CREATE EVENT:

Endpoint:

Content-Type:
  multipart/form-data (for image upload)

Parameters:

Response:

    "success": true,

⚠️ TROUBLESHOOTING:

Problem: Image not uploading
Solution:
  1. Check file size < 5MB
  2. Verify file format (JPG, PNG, GIF, WEBP)
  3. Check /uploads folder permissions
  4. Check browser console for errors (F12)

Problem: Event not appearing
Solution:
  1. Refresh page (F5)
  2. Check browser console for errors
  3. Verify all required fields filled
  4. Check database connection

Problem: "Failed to upload image"
Solution:
  1. Directory permission issue - run:
     chmod 755 c:\xampp\htdocs\EventSystem\uploads
  2. Restart Apache in XAMPP
  3. Try with smaller image file

📊 CAPACITY STATISTICS:

Query to get events with occupancy percentage:

  SELECT 
    event_name,
    capacity,
    COUNT(DISTINCT r.registration_id) as registrations,
    ROUND(100 * COUNT(DISTINCT r.registration_id) / capacity, 2) as occupancy_percent
  FROM events e
  LEFT JOIN registrations r ON e.event_id = r.event_id
  GROUP BY e.event_id;
